Array:-
Array is the collection of elements of same data type in contiguous memory locations.
Indexing starts from 0.
Example:-array.cpp

Declaration of an Array:-
We can create/declare an array by specifying its datatype then name and then using sqaure brackets "[]" and its size inside the brackets.
    eg:- int arr[5];

Initialize an array:-
Initializing means assigning values at the start.
It is done using "{}" curly brackets and inside there seperate using ",".
    eg:-int arr[5]={1,2,3,4,5};
The values will be assigned sequentially.

the size of array can be skipped if the size should be same as the number of values.
    int arr[]={1,2,3,4,5};

moreover all elements can be initialized as 0.
    int arr[5]={0}

Operations in Array elements:-
1. Access Array Elements:-
    array_name[index];
this will return the element at that particular index.
Index cannot be negative or greater than size of array.
eg:-array.cpp

2. Update Array Elements:-
    array_name[index]=value;
this will change the previous value to the new value.
eg:-array.cpp

3. Traverse Array
Traversing means visiting each element one by one.
the advantage of array is that it can be easily traversed using loops that runs from 0 to size-1.
eg:-array.cpp
