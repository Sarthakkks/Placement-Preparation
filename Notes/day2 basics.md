Control Flow Statements:-
it is a fundamental component of programming which allows developer to control the order in which the instructions will executed in a program.
They enable the execution of block of code multiple times, based on the conditions.
Types of Control Flows:-
    1.Conditional Statements:
        1.if-else(done in day1)-executes block of code if condition is true otherwise executes the another block if it is false.
        2.switch-case-Evaluates a variable or expression and executes code based on the case.
    2.Looping Statements:-
        1.for-Executes the block of code a specified times of times, typically iterating over range of values.
        2.while-Executes the program as long as condition is true.
        3.do-while-Executes the block of code once and then repeats the execution as long as condition is true.
    3.Jump Statements:-
        1.break-terminates the loop or switch statement and transfers the control to statements immediately following the loop or switch.
        2.continue-skips the current itteration and continues the next.
        3.return-Exits a function and returns a value to the caller.
        4.goto-transfers control to the labeled statement within the same function (error proned).
