Basic Input/Output in C++:
Data is written over in streams which is sequence of bytes.
    1.Input Stream:- Data given by user  to the computer memory.
    2.Output Stream:- Data from memory to user at screen.
they are defined as <iostream> header file.
    Cin>>-- for input
    Cout<<-- for output


Variables:-
They are containers used to store data in memory.
A variable has 3 components:-
        1.Data Type: Defines the kind of data stored.
        2.Variable Name: A unique identifier like num=3,here num is variable name
        3.Value: Actual data assigned to variable.

        eg:- int num=3; 

Datatypes:-
Datatypes are the type of data stored in memory, where every variable must be associated with a datatype that defines the kind of value it can store for compiler to understand.
Types of Data types:-
    1. Basic- int,float,char,double,bool,void
    2. Derived- array,pointer,reference,function
    3. User Defined- class,structure,union,typedef,using
Size of Data Types in C++
The exact size of a data type depends on the compiler and platform.

Data Type	Typical Size
char	1 byte
bool	1 byte
int	4 bytes
float	4 bytes
double	8 bytes

it can be verified using sizeof(datatype);
